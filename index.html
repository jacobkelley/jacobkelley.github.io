My name is Jacob Kelley, and I have spent most of my life working with computers, in almost every capacity.

I began as many do, by building computers and learning about computer hardware. From this point, I was lucky enough to get the advice that I should pursue the CCNA certification, which I did. After getting into college and completing the CCNA coursework, I decided that Information Security was likely to be a big part of IT in the future, so I changed my focus from Computer Network Management to Information Systems Security.



In my professional life, I have worked as a Help Desk Tech, System Administrator, General Manager, Project Manager, Pen Test Lead, NOC Technician, SOC Technician, SOC Manager, Risk Assessor, and Software Project Lead.



In programming, I primarily work in Powershell, Python, and Javascript, but I've done work with SQL and some other languages.

Please note that this is just my personal repository, and does not cover all of the code I've written. In addition to code created for work, I have also written code for CTFs that are not captured here (I took 2nd place in a 2018 OWASP CTF, and participate in the SANS Holiday Hack Challenge every year).

PowerShell

  1. PowerSteg

    GitHub

    PowerShell utility created to embed commands or C2 information in images, mainly to evade L7 firewall and AV detection.

  2. Power Shift

    GitHub

    Utility that will shift the LSB and add a file extension, to facilitate downloading binary/executable files into a network without matching IDS/IPS signatures.
    Use one of the oldest encryption techniques to evade the newest firewalls!

  3. RedVsBlue

    GitHub

    PowerShell form-based utility to evaluate local system's incident preparedness, improve logging locally, as well as deploy scripts to protect organizational assets.

  4. Monitor Outbox

    GitHub

    MonitorOutbox.ps1 is a utility that constantly scans your Outlook outbox, and creates a popup message if there are any unsent emails in it.

  5. Get Wallpapers

    GitHub

    A utility that uses Reddit to download new wallpapers, and rotate through them. Created as a compliment to PowerSteg, for use as a large image repository that would enable execution of large commands and C2 functions.

  6. scanServices

    GitHub

    Quick utility for checking what services are running on all systems in an OU. Use to scan for service account configs for resetting passwords.

  7. Server Service Scanner

    GitHub

    Scan a number of servers for non-Windows Server services, with a switch to configure known services as well, to ignore those if you only want to look for suspicious services installed on your servers.

  8. Event Review

    GitHub

    Find critical and warning events, and output to a gridview.

  9. Sinkhole

    GitHub

    Download popular malware lists and re-route traffic to the bitbucket to prevent damage from being done if you get malware.

  10. grayb0x

    GitHub

    This utility is intended for security analysts on whitebox(blue team) and blackbox(red team) engagements.
    It is broken out into local and network functionality to provide control over SOC visibility into activities being conducted.

  11. PII Scanner

    GitHub

    Recursively look for interesting strings in documents.
